# Koushik Murali - Portfolio

A modern, interactive personal portfolio website built to showcase my projects, skills, and professional experience in AI and Data Science. This project leverages 3D graphics, smooth animations, and a responsive design to provide an engaging user experience.

## Features

- **Hero Section:** Immersive 3D astronaut scene with parallax background effects.
- **About Me:** Interactive grid layout highlighting personal details, tech stack, and a 3D globe visualization.
- **Projects Showcase:** Detailed view of my key projects (Deepfake Detection, Bug Detection, HSN Code Validation) with links to repositories.
- **Experience Timeline:** A visual timeline of my professional journey and internships.
- **Contact Form:** Fully functional contact form integrated with EmailJS for direct communication.
- **Responsive Design:** Optimized for all devices, from mobile phones to large desktop screens.

## Installation & Running

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/koushik-777/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```

4.  **Build for production:**

    ```bash
    npm run build
    ```

5.  **Preview the production build:**
    ```bash
    npm run preview
    ```
